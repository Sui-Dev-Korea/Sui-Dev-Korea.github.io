## Sui RPC

Sui 블록체인을 구동하는 코드에 대한 자세한 내용은 Sui 프레임워크 및 Sui RPC 문서를 참조하라.

<cards></cards>

<card title="GraphQL for Sui RPC" href="/references/sui-graphql"></card><card title="JSON-RPC" href="/references/sui-api"> 새로운 프로젝트의 경우 Sui RPC에 GraphQL을 사용한다. 아직 GraphQL로 마이그레이션하지 않은 기존 프로젝트의 경우 JSON-RPC 참조를 사용한다.</card>

## Move

Move는 Sui 블록체인의 스마트컨트랙트 로직을 구현하는 데 사용된다. Move에 대해 더 자세히 알아보거나 복습하려면 다음 자료를 활용하자.

<cards><card title="Sui framework" href="/references/framework"></card><card title="The Move Book" href="https://move-book.com/"> Move Book은 Sui 블록체인 상의 Move 프로그래밍 언어에 대한 종합적인 안내서이다.</card><card title="The Move Reference" href="https://move-book.com/reference/"> Move 참조 문서는 Move 프로그래밍 언어의 아키텍처와 구문을 설명한다.</card></cards>

## Sui CLI

Sui 명령줄 인터페이스(CLI)를 사용하여 Sui 네트워크 및 해당 기능과 직접 상호 작용할 수 있다. CLI는 특정 기능 세트를 대상으로 하는 여러 기본 명령으로 구성된다.

<cards><card title="Sui Client CLI" href="/references/cli/client">Sui Client CLI를 사용하여 Sui 네트워크에 클라이언트를 생성하고, 주소를 생성하고, 네트워크에 액세스하는 등의 작업을 수행할 수 있다.</card><card title="Sui Client PTB CLI" href="/references/cli/ptb"> Sui Client PTB CLI를 사용하면 터미널에서 직접 프로그래밍 가능한 transaction 블록을 구축, 미리 보기 및 실행할 수 있다.</card><card title="Sui Move CLI" href="/references/cli/move"> Sui Move CLI를 사용하여 체인에서 Sui Move 기능에 액세스해보자.</card></cards>

<card title="Sui Replay CLI" href="/references/cli/replay">Sui Move CLI를 사용하여 체인에서 Sui Move 기능에 액세스하자.</card>

## Sui software development kits

Sui에서 사용할 수 있는 공식 소프트웨어 개발 키트(SDKs)에는 TypeScript SDK와 Rust SDK가 있다.
