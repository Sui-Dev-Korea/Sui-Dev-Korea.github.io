Sui의 모든 것은 object이며 스마트 컨트랙트는 필연적으로 네트워크에서 해당 object를 한 소유자에게서 다른 소유자로 이동(전송)해야 한다. 이 섹션의 주제들은 네트워크에서 object를 전송하는 것과 관련하여 Sui에서 사용할 수 있는 선택지를 탐구한다.
