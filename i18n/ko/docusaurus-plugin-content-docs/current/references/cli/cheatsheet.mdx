## Addresses &amp; aliases

<table class="w-100">
	<thead>
		<tr>
			<td>Command</td>
			<td>Description</td>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td class="w-2/3">`sui client active-address`</td>
			<td class="w-1/3">활성 주소를 가져온다</td>
		</tr>
		<tr>
			<td class="w-2/3">`sui client addresses`</td>
			<td class="w-1/3">주소, 별칭 및 현재 활성 주소를 나열한다.</td>
		</tr>
		<tr>
			<td class="w-2/3">`sui client new-address ed25519`</td>
			<td class="w-1/3">ED25519 체계를 사용하여 새 주소를 생성한다.</td>
		</tr>
		<tr>
			<td class="w-2/3">`sui client new-address ed25519 MY_ALIAS`</td>
			<td class="w-1/3">ED25519 체계와 별칭으로 새 주소를 만든다.</td>
		</tr>
		<tr>
			<td class="w-2/3">`sui client switch --address ADDRESS`</td>
			<td class="w-1/3">이것을 활성 주소로 만든다(별칭도 허용).</td>
		</tr>
		<tr>
			<td class="w-2/3">`sui keytool convert PRIVATE_KEY`</td>
			<td class="w-1/3">Hex 또는 Base64 형식의 개인 키를 새로운 형식으로 변환한다(Bech32 인코딩된 33바이트 플래그 || "suiprivkey"로 시작하는 개인 키)</td>
		</tr>
		<tr>
			<td class="w-2/3">`sui keytool generate ed25519`</td>
			<td class="w-1/3">ED25519 체계를 사용하여 새 키 쌍을 생성하고 파일에 저장한다.</td>
		</tr>
		<tr>
      <td class="w-2/3">`sui keytool import INPUT KEY_SCHEME`</td>
			<td class="w-1/3">입력 니모닉 구문 또는 Bech32로 인코딩된 33바이트 플래그 || "suiprivkey"로 시작하는 privkey를 사용하여 Sui CLI 키 저장소에 새 키를 추가한다.</td>
		</tr>
		<tr>
			<td class="w-2/3">`sui keytool update-alias OLD_ALIAS NEW_ALIAS`</td>
			<td class="w-1/3">주소의 별칭을 업데이트한다</td>
		</tr>
	</tbody>
</table>

## Faucet &amp; gas

<table class="w-100">
	<thead>
		<tr>
			<td>Command</td>
			<td>Description</td>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td class="w-2/3">`sui client faucet`</td>
			<td class="w-1/3">활성화된 네트워크와 연결된 faucet에서 SUI 코인을 얻는다.</td>
		</tr>
		<tr>
			<td class="w-2/3">`sui client faucet --address ADDRESS`</td>
			<td class="w-1/3">해당 주소로 SUI 코인을 받는다 (별칭도 허용됨).</td>
		</tr>
		<tr>
			<td class="w-2/3">`sui client faucet --url CUSTOM_FAUCET_URL`</td>
			<td class="w-1/3">커스텀 faucet에서 SUI 코인을 받는다</td>
		</tr>
		<tr>
			<td class="w-2/3">`sui client gas`</td>
			<td class="w-1/3">활성 주소에 대한 가스 코인을 나열</td>
		</tr>
		<tr>
			<td class="w-2/3">`sui client gas ADDRESS`</td>
			<td class="w-1/3">주어진 주소에 대한 가스 코인 목록을 표시한다(별칭도 허용).</td>
		</tr>
	</tbody>
</table>

## Create, build, and test a Move project

<table class="w-100">
	<thead>
		<tr>
			<td>Command</td>
			<td>Description</td>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td class="w-2/3">`sui move build`</td>
			<td class="w-1/3">현재 디렉토리에 Move 프로젝트를 빌드한다</td>
		</tr>
		<tr>
			<td class="w-2/3">`sui move build --path PATH`</td>
			<td class="w-1/3">제공된 경로에서 Move 프로젝트를 빌드한다</td>
		</tr>
		<tr>
			<td class="w-2/3">`sui move migrate PATH`</td>
			<td class="w-1/3">제공된 경로의 패키지에 대해 Move 2024로 마이그레이션한다.</td>
		</tr>
		<tr>
			<td class="w-2/3">`sui move new PROJECT_NAME`</td>
			<td class="w-1/3">지정된 폴더에 새 Move 프로젝트를 생성한다.</td>
		</tr>
		<tr>
			<td class="w-2/3">`sui move test`</td>
			<td class="w-1/3">현재 디렉토리에서 Move 프로젝트를 테스트한다.</td>
		</tr>
		<tr>
			<td class="w-2/3">`sui move test --trace`</td>
			<td class="w-1/3">현재 디렉터리의 Move 테스트에 대한 실행 트레이스를 생성한다. [Move Trace Debugger](https://marketplace.visualstudio.com/items?itemName=mysten.move-trace-debug) 확장 프로그램과 함께 사용한다.</td>
		</tr>
	</tbody>
</table>

## Executing transactions

<table class="w-100">
	<thead>
		<tr>
			<td>Command</td>
			<td>Description</td>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td class="w-2/3">`sui client call \`<br>  `--package PACKAGE \`<br>  `--module MODULE \`<br>  `--function FUNCTION`</td>
			<td class="w-1/3">Move 패키지를 호출한다</td>
		</tr>
		<tr>
			<td class="w-2/3">`sui client merge-coin \`<br>  `--primary-coin COIN_ID \`<br>  `--coin-to-merge COIN_ID`</td>
			<td class="w-1/3">두 코인을 merge한다.</td>
		</tr>
		<tr>
			<td class="w-2/3">`sui client split-coin \`<br>  `--coin-id COIN_ID \`<br>  `--amounts 1000`</td>
			<td class="w-1/3">코인을 두 개로 나눈다. 하나는 1000 MIST가 들어있고 나머지는 다른 하나에 있다</td>
		</tr>
		<tr>
			<td class="w-2/3"> 				`sui client pay-sui \`<br>  `--input-coins COIN_ID \`<br>  `--recipients ADDRESS \`<br>  `--amounts 100000000`</td>
			<td class="w-1/3">0.1 SUI를 주소로 전송하고 동일한 코인을 가스로 사용한다.</td>
		</tr>
		<tr>
			<td class="w-2/3">`sui client transfer-sui \`<br>  `--sui-coin-object-id COIN_ID \`<br>  `--to ADDRESS`</td>
			<td class="w-1/3">SUI object를 주소로 전송하고 동일한 코인을 가스에 사용한다.</td>
		</tr>
	</tbody>
</table>
