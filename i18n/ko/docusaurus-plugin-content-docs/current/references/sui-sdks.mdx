Sui는 Sui API의 wrapper로 동작하는 개발자 키트를 제공한다. Sui 커뮤니티는 Sui 블록체인을 대상으로 하는 자체 개발자 키트를 통해 코드 커버리지를 확장한다.

## Sui SDKs

<cards> <card title="dApp Kit" href="https://sdk.mystenlabs.com/dapp-kit">     Sui API와 상호작용하는 웹 프론트엔드 SDK이다. NPM 패키지로 제공된다.</card> <card title="Rust SDK" href="./rust-sdk">     `sui-sdk` crate를 사용하여 Rust로 Sui API를 사용하는 방법과 SDK 설정 예제를 제공한다. </card> <card title="Rust SDK Auto-generated Docs" href="https://mystenlabs.github.io/sui/sui_sdk/index.html">     Sui 리포지토리 내 `sui-sdk` crate에 대한 자동 생성 문서이다. </card> <card title="TypeScript SDK" href="https://sdk.mystenlabs.com/typescript">     TS dApp에 Sui를 통합하기 위한 TypeScript SDK이다. </card> <card title="zkSend SDK" href="https://sdk.mystenlabs.com/zksend">     Stashed 기능을 통합할 수 있도록 하는 zkSend SDK이다. </card> </cards>

## Community SDKs

:::info

커뮤니티 프로젝트들은 전문적으로 개발되었지만, 유지보수 및 커뮤니티 지원 수준은 다양하다. 유틸리티 사용을 결정하기 전에 프로젝트의 이력과 지원 수준을 조사하는 것이 좋다.

:::

<cards> <card title="dApp Kit (Vue)" href="https://github.com/SuiFansCN/suiue">      Vue 프레임워크용 Sui dApp Kit이다. </card> <card title="Dart SDK" href="https://github.com/mofalabs/sui">     모바일, 웹, 데스크톱용 크로스 플랫폼 Sui SDK이다. </card> <card title="Go SDK" href="https://github.com/block-vision/sui-go-sdk">     Golang을 사용하여 Sui용 애플리케이션을 개발하기 위한 SDK이다. </card> <card title="Kotlin SDK" href="https://github.com/mcxross/ksui">     Ksui는 Sui Full Node와 상호작용하기 위한 Kotlin Multiplatform JSON-RPC wrapper 및 암호화 유틸리티 컬렉션이다. </card> <card title="Python SDK" href="https://github.com/FrankC01/pysui">     pysui는 Sui 블록체인에서 개발하기 위한 Python 클라이언트이다. </card> <card title="Swift SDK" href="https://github.com/opendive/suikit">     SuiKit은 Sui 블록체인 개발을 위해 네이티브로 설계된 Swift SDK이다. </card> </cards>
